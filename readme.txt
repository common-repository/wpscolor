== Description ==
wpscolor is a Color Switcher and Theme Option Color Settings plugin with Sidebar Presentation for WordPress. You can use this plugin to customize the color scheme of any WordPress theme, implement a front-end color switcher, and enhance front-end theme sidebar presentation.

This section describes how to install the plugin and get it working Best Way.
e.g.
1. Upload `wpscolor` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
